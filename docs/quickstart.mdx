## Overview

Brane is a modern, type-safe Java SDK for Ethereum. It provides a clean, fluent API for interacting with EVM chains, with a focus on correctness and performance.

## Installation

<Warning>
**Private Access Only**

Brane is currently in **private alpha**. The artifacts are not yet available on Maven Central.
</Warning>

### For Maintainers

If you are contributing to Brane or have access to the source code, you can install the SDK locally.

1.  Clone the repository:
    ```bash
    git clone https://github.com/noise-xyz/brane.git
    cd brane
    ```

2.  Publish to local Maven:
    ```bash
    ./gradlew publishToMavenLocal
    ```

3.  Add `mavenLocal()` to your project's repositories:

    <CodeGroup>
    ```xml Maven
    <repositories>
        <repository>
            <id>local-maven</id>
            <url>file://${user.home}/.m2/repository</url>
        </repository>
    </repositories>
    ```

    ```groovy Gradle
    repositories {
        mavenLocal()
        mavenCentral()
    }
    ```
    </CodeGroup>

### Dependencies

Once configured, add the dependencies:

<CodeGroup>

<dependencies>

## Usage

### 1. Connect to a Chain

Create a `PublicClient` to read from the blockchain.

```java
import io.brane.rpc.BranePublicClient;
import io.brane.core.chain.ChainProfiles;

public class Example {
    public static void main(String[] args) {
        // Connect to Mainnet
        var client = BranePublicClient.forChain(ChainProfiles.MAINNET)
            .build();
        
        // Get the latest block number
        var block = client.getLatestBlock();
        System.out.println("Current Block: " + block.number());
    }
}
```

### 2. Send a Transaction

Create a `WalletClient` to sign and send transactions.

```java
import io.brane.rpc.BraneWalletClient;
import io.brane.core.types.Wei;
import io.brane.core.types.Address;

// ...

var wallet = BraneWalletClient.forChain(ChainProfiles.SEPOLIA)
    .withPrivateKey(System.getenv("SEPOLIA_PRIVATE_KEY")) // Load private key from environment variable
    .build();

var receipt = wallet.sendTransactionAndWait(
    new TransactionRequest(
        wallet.getAddress(),           // from
        new Address("0x..."),          // to
        Wei.fromEther("0.01"),         // value
        null,                          // gasLimit (auto-estimated)
        null,                          // gasPrice (not used for EIP-1559)
        null,                          // maxPriorityFeePerGas (auto-estimated)
        null,                          // maxFeePerGas (auto-estimated)
        null,                          // nonce (auto-fetched)
        null,                          // data (none for simple transfer)
        true,                          // isEip1559
        null                           // accessList
    ),
    60_000, // timeout (ms)
    1_000   // poll interval (ms)
);

System.out.println("Tx Hash: " + receipt.transactionHash());
```
