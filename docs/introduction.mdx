## Welcome to Brane

Brane is a high-performance, type-safe Java SDK for interacting with Ethereum and other EVM-compatible blockchains. Built on Java 21, it focuses on clarity, correctness, and developer experience.

## Features

- **Type-Safe Contract Binding**: Interact with contracts using standard Java interfaces.
- **Modern Transactions**: EIP-1559 support by default with smart gas estimation.
- **EIP-4844 Blob Transactions**: Send data availability blobs for rollups with KZG commitment support.
- **Zero-Codegen**: Runtime binding speeds up iteration.
- **Clean API**: Minimal surface area with no legacy dependencies.
- **First-Class Testing**: `Brane.Tester` provides snapshots, impersonation, and time manipulation for integration tests.
